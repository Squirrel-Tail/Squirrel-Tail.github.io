写作技巧 Link to heading

写作，就是支撑写作本身的唯一技巧。

过去几年，我曾一度沉迷所谓“生产力方式”。期间应该看过了不下几十本关于写作的书籍，得到收获最多的还是开始写之后。

那么知道这些技巧还有帮助么？

我认为是有的，至少可以让人起步阶段的煎熬少点，毕竟基本所有作家和写作者都有过枯坐几小时写不出东西的时刻。

所以本期我整理了一些“大师的写作方式”与“小众写作方法”。当然不会是全部，以后有机会还会更新更多写作技法。

大师的写作方法 Link to heading

  • 乔治马丁 代表作《冰与火之歌：权力的游戏》
    • 名称：“建筑师”与“园丁”型作家
    • 很早就听说过这种说法，我找过出处，发现作者在很多访谈中都谈论过这个说法，但通常就一两段简短的内容。后来，机缘巧合发现一版较少出现的完整版本。
    • 出自：《奇迹之书》最后的附录——《乔治·R．R．马丁谈写作技艺》
    • 作者：杰夫范德米尔
    • 地址：实体书
    • 推荐度：5颗星
    • 简介：
      • 大约十页的一次访谈记录。从初稿到创作过程，从早期作品到最热门的大作，都有所涉及。并且马丁也提到了“海因莱茵的写作法则”！
      • 由于提问者本人也是非常著名的作家（《遗落的南境:湮灭》的作者）所以许多问题都切中要害。
      • 比如“你很擅长场景…所以…”、“《冰与火之歌》电视剧版里有什么场景让你想要回去修改原著？”、“有没有哪种场景是你特别不想写的？”等场景相关的数个问题。
        • 对于最后一个问题，马丁的回答很有趣。权力的游戏第一部中有三场战役相隔很近，他的处理是：第一场战役详细描述，充满画面感。第二场战役用一个未参与战斗角色的视角进行描述。（她在一座山上俯瞰战场，并且看不到战斗，只能听到音效。）第三场战役则只让一名信使讲述战斗经过。
      • 其余还有一些设定相关的内容。
      • 最后是关于对新手作家的建议：
        • 针对编辑提出修改故事的要求，马丁的建议是：“假如你觉得他人的意见会削弱故事，就绝对不要改，因为最终人们将根据故事的水准来评判你。你不能申辩说‘是编辑让我改的。’”
  • 斯蒂芬金 代表作《闪灵》《迷雾》《肖申克的救赎》《绿里奇迹》…
    • 名称：斯蒂芬金创作生涯回忆录
    • 出自：《写作这回事》
    • 作者：斯蒂芬金
    • 地址：微信读书（付费）/实体书
    • 推荐度：4颗半星
    • 简介：
      • 天赋型选手的创作感想。
      • 前半部分是作者本人回忆录，后半部分是写作指导。
      • 这本书有意思的地方正在于此，你能从前半部分看出作者的写作能力，那些很妙的比喻，叙事手法，如何吸引读者读下去，如何把普通人的日常写得有趣，还有作者的心态，愿景，经历…后半部分反而没那么重要了。
      • 作者提到词汇、语法、句子层面的内容。
      • 作者认为想成为作家就必须“多读，多写。”
      • 作者着墨写作心态，通过自己的经历真诚的表达。
      • 作者推荐“关上门来写作，敞开门来修改。”
      • 作者将自己的写作比喻为“考古发掘”。
        • “我相信故事如埋在地下的化石，是被人们发掘出来的。”“它们是遗迹，属于一个未被发现但已经存在的世界。作家的工作就是利用他/她工具箱里的工具，把每个故事尽量完好无损地从地里挖出来。”
      • 作者写作通常是先想一个有趣的开头，然后打开“他妈的”打字机，就往下面敲啊！

小众写作方法： Link to heading

  • 名称：雪花写作法
    • 出自：《雪花写作法:10步写出一篇好小说》
    • 作者：兰迪·英格曼森
    • 地址：微信读书（免费）
    • 简介：
      • 作者使用数学中的“雪花分形”比喻来创作小说。
      • 整本书本身也是一个故事，并且将雪花写作法方法融入其中。
      • 雪花写作法从“一句话故事”开始创作，这也是许多剧本创作中的手法。（有些人没有时间看长篇大论）
  • 名称：九宫格写作法
    • 出自：《九宫格写作法》
    • 作者：山口拓朗
    • 地址：微信读书（付费）/自行下载
    • 简介：
      • 日本人最喜欢写的这种“操作手册”形式的工具书籍。
      • 包含素材收集、如何写作与写作模板等部分
      • 像一篇篇公众号凑成的书
  • 名称：创意写作
    • 出自：创意写作这个名词国内提得不多。有一套创意写作书系一共几十本，此处列出其中一本：《小说写作教程：虚构文学速成全攻略》
    • 作者：杰里·克利弗
    • 地址：微信读书（付费）/自行下载
    • 简介：
      • 创意写作书籍中最为实操的书之一。
      • 作者是教创意写作的老师，这本书有点像是课本，每章后面还有习题。
  • 名称：写作脑科学
    • 地址：微信读书（付费）/实体书
    • 作者：屠龙的胭脂井
    • 简介：
      • 如同标题，分为写作和脑科学。脑科学部分基本都是牵强附会，噱头碾压实际，毫无用处。
      • 写作部分，道理还是那些道理，比如拆解-建构，积累素材，组织素材，虽然作者希望扯上建构主义的大旗，但还是过于浅显了。
      • 但有意思的地方在于，作者是写微博的，所以本书阅读压力基本为0，作者也在本书讨论了这点。
      • 并且作者给了一套快速成文的方法论——“快速写作，宏观写作，微观写作”，这点值得肯定。
      • 作为现代中国的作者，许多观点并不陈旧乡土，但又遗传自古代文脉，比如关于韵律和代入感的讨论。
      • 而且作者有西方留学的经历，有些方法论明显是取自西方剧作法，比如“不同文化，共同品味”的章节部分。
      • 这就让这本书“麻雀虽小五脏俱全”，我个人认为还是本不错的实践指南。
      • 适合高考前学生，或者初写文字者。
  • 名称：小说课
    • 地址：微信读书（免费）
    • 作者：毕飞宇
    • 简介：
      • 作者是选出中外经典作品进行讲解，比如《水浒》、《红楼梦》、《项链》等
      • 这是一套作者音频配套讲稿。（但是现在音频我找不到了）
      • 因为是音频讲稿，所以比较好读。
      • 作者本人是作家，所以对于文学读的很深很细，讲解也偏向细节方面。大致是告诉读者，为何名作是名作的意味。
      • 印象深刻的是作者对于水浒、杀手、受戒的分析比较有意思。
      • 作者不懂英文，但是说自己能通过翻译的文本读懂原著作者的意思，据考，并无任何说服力。

最后还有一个提醒，许多“大师的技巧”是有冲突的，比如最经典的托尔金写《指环王》的时候反复修改，字斟句酌，一本书写了十几年。而斯蒂芬金推荐的写作方式则是：想好一个有意思的开头，就拿起“他妈的”打字机就是个写啊！

我比较认同的还是中道的方式：即上期介绍的——海因莱茵的写作法则
